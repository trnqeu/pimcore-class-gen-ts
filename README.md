# pimcore-class-gen-ts

## Installazione

Installazione dipendenze

```bash
npm install
```

## Utilizzo

- Ripulire il csv delle righe duplicate su Google Spreadsheet
- Lanciare lo script
```
npx tsc
node dist/main.js ./csv/esempio.csv
```
- Importare i tracciati
- Inserire i localized fields
- Gestire liste multiselect
- Cambiare da textarea a WYSIWYG quando serve
